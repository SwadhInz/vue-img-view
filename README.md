# vue-img-view

> Vue 图片组件 (缩放,旋转,拖动)

## DEMO
[DEMO](https://la7ender.github.io/vue-img-view/ "demo")
## Install

``` bash
# install
npm install vue-img-view
```


## Setup

``` javascript
import imgView from 'vue-img-view'
Vue.use(imgView)
```


## Use

``` html
<img-view :imgUrl="imgUrl" :background="background"></img-view>
```
