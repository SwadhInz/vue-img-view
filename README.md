# vue-img-view

> Vue 图片组件
>  npm install vue-img-view
>  import imgView from 'vue-img-view'
>  Vue.use(imgView)

> <img-view :imgUrl="imgUrl" :background="background"></img-view>


## Install

``` bash
# install
npm install vue-img-view
```


## Setup

``` javascript
import imgView from 'vue-img-view'
Vue.use(imgView)
```


## Use

``` html
<img-view :imgUrl="imgUrl" :background="background"></img-view>
```
